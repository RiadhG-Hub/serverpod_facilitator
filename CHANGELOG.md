## 1.0.2

- Enhanced CLI to display help text and all available commands when run without arguments.
- Improved `ai` command to generate models following Google/Dart (Effective Dart) style guidelines, including documentation comments.

## 1.0.1

- Renamed the CLI executable to `serverpod_facilitator` for better consistency.
- Added support for global installation via `dart pub global activate`.
- Updated all documentation and help messages to use the new standardized command name.
- Improved consistency in generated model code for projects created with `create`.

## 1.0.0

- Initial release of `serverpod_facilitator`.
- Support for AST-based Dart model parsing.
- Automatic Serverpod YAML schema generation.
- PostgreSQL-aware annotations for fine-grained database control.
- SQL migration engine with preview and diff capabilities.
- Integrated AI-powered model generator.
- CLI tools for rapid bootstrapping and code generation (Auth, Admin, Client).
